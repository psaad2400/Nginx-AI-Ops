# Grafana Dashboard

This project uses the official **Node Exporter Full** dashboard.

## Import Instructions

```
Grafana → Dashboards → New → Import
Dashboard ID : 1860
Data Source  : select your Prometheus
→ Import
```

Official dashboard: https://grafana.com/grafana/dashboards/1860
